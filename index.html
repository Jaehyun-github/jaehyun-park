Olentia(개인 프로젝트)

  • Visual Studio Code
  • HTML, CSS, JavaScript
  • 사용한 라이브러리 : JQuery, AOS

개인 프로젝트로 처음 만들어 본 디퓨저 판매 사이트 "Olentia"입니다.
웹사이트의 메인화면은 각각 섹션 주제에 맞게 동영상을 첨부했습니다.
(ex. SLEEPING 섹션에서 아이가 잠든 모습)
Olentia는 라틴어로 "향기"라는 뜻입니다.
제가 평소 디퓨저를 좋아하기도 하고, 코딩을 배우게 된다면
한 번쯤 만들어 보고 싶었던 웹사이트였습니다.
첫 개인 프로젝트라 다소 어설프지만 앞으로도 꾸준히 업데이트할 예정입니다.

ParisBaguette(팀 프로젝트)

  • Visual Studio Code
  • HTML, CSS, JavaScript
  • 사용한 라이브러리 : JQuery, Slick

팀 프로젝트로 리뉴얼 해보는 "ParisBaguette" 웹사이트입니다.
기존 웹사이트를 팀원과 함께 분석하며 불편한 점과 개선해야 할 것들을 찾아
리뉴얼의 방향을 잡았습니다. 첫 팀 프로젝트를 하면서 팀원들과 함께 의견을
나누고 어떻게 UIUX를 구성해야 웹사이트를 이용하는 사용자가 만족할 수
있을지 많은 고민을 했습니다. 첫 팀프로젝트를 통해 팀원들과 협업하며 개발자 환경에 적응할 수 있는 계기가 되었습니다.

NewBalance(개인 프로젝트)

  • Visual Studio Code
  • React, CSS
  • 사용한 라이브러리 : Swiper

React를 독학하면서 리뉴얼 해본 "NewBalance" 웹사이트입니다.
data.json 파일을 만들어서 fetch를 통해 데이터를 불러오고 usestate로
코드를 작성하였습니다. React는 컴포넌트를 구성하면 프로젝트 전체의
가독성이 높아지고, 간편하게 UI 수정을 할 수 있으며 컴포넌트의 재사용이 가능하기 때문에 코드를 작성했을 때 빠르고 효율적으로 화면을 구성할 수 있습니다. 이런 REACT의 매력을 느껴 저는 개발자 성장에 한걸음 더 다가갈 수 있다고
느꼈습니다.
